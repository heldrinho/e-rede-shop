# 🛍️ E-Rede Shop

E-commerce fullstack moderno construído com **Vue.js** e **Node.js**.

## 🚀 Tecnologias

**Frontend**
- Vue.js 3
- TypeScript
- HTML5 / CSS3

**Backend**
- Node.js
- Express

## 📁 Estrutura do Projeto
```
e-rede-shop/
├── 1-frontend/   # Aplicação Vue.js
└── 2-backend/    # API Node.js
```

## ⚙️ Como rodar localmente
```bash
# Clone o repositório
git clone https://github.com/heldrinho/e-rede-shop.git

# Frontend
cd 1-frontend
npm install
npm run dev

# Backend
cd 2-backend
npm install
npm run dev
```
